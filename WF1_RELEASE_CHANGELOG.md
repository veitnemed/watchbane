# Watchbane: итог WF-01—WF-60 и release-аудита

Актуальная версия: **Watchbane 0.1.1-alpha.1 — Open Route**; алгоритм выдачи: **ReDeck v0.1.0**. Базовый цикл WF-01—WF-60 закрыт 13 июля 2026; сетевое release-hardening обновлено 14 июля 2026. Этот файл фиксирует выполненные изменения и проверки от первого UI-прохода до финального release-аудита. Временные PNG, тестовые профили и диагностические логи в репозиторий не включены.

## Exploratory bug hunt BH-61—BH-75

Проведено 15 независимых циклов на чистых, больших, частично повреждённых и мигрированных профилях. Проверены быстрые действия Recommendations, Collection, dialogs, restart, offline, Unicode FTS, Saved/Hidden, onboarding, 150% scale и packaged EXE.

Найден один P1-дефект: при закрытии после underfilled refill мог разрушаться активный Qt worker. Shutdown теперь отменяет и дожидается worker, а обработка deferred-delete ограничена worker-объектами Watchbane. Повторные native и packaged запуски завершаются с exit code 0. Оставшихся P0/P1/P2 по этим циклам нет.

## Дополнительные исправления после аудита

- Устранено краткое появление окна ввода TMDb token при быстром успешном startup-path.
- Сохранённые фильтры теперь определяют стартовую deck; после перезапуска не показывается старая подборка другого intent.
- Счётчик запаса и заголовок рекомендаций не обрезаются на 150%.
- Поле `Рекомендация: …` в detail-карточке увеличено, усилено по контрасту и выровнено относительно TMDb ring.
- Recommendation strength скрыта у уже просмотренных карточек.
- Дублирующая строка `Тип` удалена из «Основной информации»; тип остаётся виден в poster badge.
