## Ручной чеклист (5–10 мин)

Предусловия: токен TMDb, onboarding пройден или скипнут.

1. [ ] Запуск → без лишних экранов попадаешь в приложение.
2. [ ] Сразу вкладка **Рекомендации** (не Коллекция / не Настройки).
3. [ ] Справа/в основной области: список с постерами **или** одно понятное «готовим».
4. [ ] Список конечный (есть «Всего N»), скролл не тянет бесконечность.
5. [ ] На выбранной карточке доступны: оценка (смотрел), «+ Запомнить», «× Не показывать».
6. [ ] Есть способ запросить ещё порцию («Обновить» / «Обновить колоду») без блуждания по advanced.

**Не требуется для C1-02:** прятать вайб-крутилки (C1-04), убирать «Поиск настроек» из daily path (C1-03), жёстко 10 карточек (C1-05).
