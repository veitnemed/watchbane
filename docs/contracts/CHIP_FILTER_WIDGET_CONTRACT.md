# Контракт виджета chip filter

Контракт относится к selector-ам жанров и стран в desktop GUI.

## Источник порядка

- Service/data layer отдаёт жанры и страны уже отсортированными по популярности: чем больше тайтлов у жанра/страны, тем выше элемент.
- UI-widget не пересчитывает популярность и не меняет базовый порядок данных.

## Текущий порядок в виджете

При каждом изменении выбора selector строит текущий порядок так:

1. выбранные элементы;
2. невыбранные элементы.

Внутри каждой группы сохраняется исходный порядок популярности.

Пример:

```text
base:      A, B, C, D, E, F, G
selected:  F, C
display:   C, F, A, B, D, E, G
```

## Collapse / Expand

- Collapsed mode показывает строго первые 5 элементов текущего порядка.
- Expand mode показывает все элементы.
- Выбранные элементы не добавляются отдельным хвостом поверх лимита.
- Если выбрано больше 5 элементов, в collapsed mode видны только первые 5 выбранных по популярности.
- Кнопка «Показать еще» показывает количество реально скрытых элементов.

## Empty State

- Пустой список жанров/стран не должен падать.
- Expand-кнопка скрыта, если элементов 5 или меньше.

## Запреты

- Не менять порядок случайно при check/uncheck.
- Не показывать дополнительные chips вне лимита collapsed mode.
- Не сортировать выбранные элементы по времени клика.
- Не пересчитывать популярность внутри UI-widget.
