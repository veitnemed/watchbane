# Archive

Здесь лежат неактивные части старой версии проекта.

- `legacy/model/` — старая ML-модель, обучение, LOO-отчеты и диагностика. Активный runtime `series-list` больше не импортирует этот код.
- `legacy/tests/` — старый монолитный тестовый набор. Активный pytest-набор живет в `tests_pytest/`.

Архив нужен только как временная страховка при чистке структуры. Новый код сюда не добавлять.
