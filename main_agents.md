- одна задача = один ID из PRODUCT_ROADMAP;
- commit только по просьбе пользователя.

Дополнительные UI-привычки (layout, QSS, screenshot tiers) при необходимости смотри историю git для прежней длинной версии `main_agents.md` или следуй `DESKTOP_STYLE_CONTRACT` и корневому `AGENTS.md`.
